About the project

This is my project for the individual Project Assignment for the Spring Advanced Course at SoftUni. I chose this topic because I’ve always been fascinated by the wonders of the universe. The project aims to take the user on an adventure around our solar system and introduce him to the captivating world that lays beyond our planet. Once the user registers, he will have access to all the community’s posts, and you will be able to post too. As the name of the projects suggests, I will continue expanding it and add new information and features in the future. Let me know if you have any questions or suggestions. I hope you like my little project and hopefully you learn something new.
